patternshare
============
内部分享的平台

#要求：
  Nodejs last version（v0.10.13）
  mongodb last version（2.4.7）

#安装：
  先启动mongodb (cd pathto/mongodb/bin && sudo ./monogod),如有问题参照提示sudo mkdir /data/db。
  启动项目(cd pathto/project/ &&　node install && node app.js)
  浏览器访问（http://127.0.0.1:3000）
